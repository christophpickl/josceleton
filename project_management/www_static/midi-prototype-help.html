You kneed help?

Sample:

r_hand, X, [0.0 .. 1.0 => 0 .. 127], 1, 1

Format:

Joint, Direction, Range, MIDI Channel, MIDI Controller Number
  1. Joint: any of the following
    can be set relative to another joint by eg: r_hand#torso
    • head
    • neck
    • torso
    • r_hand
    • l_hand
    • r_shoulder
    • l_shoulder
    • r_elbow
    • l_elbow
    • r_shoulder
    • l_shoulder
    • r_hip
    • l_hip
    • r_knee
    • l_knee
    • r_foot
    • l_foot
  2. Direction: one of { X, Y, Z }
  3. Range:format fromStart .. fromEnd => toStart .. toEnd
  4. MIDI Channel: one of { 1-16 }
  5. MIDI Controller Number: one of { 1-16 }
